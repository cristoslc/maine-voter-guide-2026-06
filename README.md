# Maine Voter Guide — South Portland, June 2026

A nonpartisan voter's guide for South Portland residents in the June 2026 Maine election.

## What's Inside

- Candidate profiles and positions for contested races
- Ballot question explanations with pro/con summaries
- Polling locations and hours for South Portland
- Key dates: voter registration deadline, early voting, election day
- Links to official Maine Secretary of State resources
